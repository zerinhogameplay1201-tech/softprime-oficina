# Softpime Oficina (MVP)

Mini-aplicação para Auto Elétrica (clientes, veículos, serviços, agendamentos).

Requisitos:
- Node.js 16+ instalado
- npm

Como rodar:
1. No terminal, dentro da pasta do projeto (softpime-oficina):
   - npm install
2. Iniciar:
   - npm start
3. Abrir no navegador:
   - http://localhost:3000

Notas:
- Banco SQLite será criado em ./data/softpime.db automaticamente.
